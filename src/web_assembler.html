Custom 16-Bit CPU Web Assembler

Quick Start Guide

  • Write Code: Type your custom assembly in the left panel. Registers are R0 to R7.
  • Use Labels: Define jump targets by ending a word with a colon (e.g., LOOP:). The assembler calculates the exact instruction address automatically.
  • Assemble: Click "Assemble Code" to parse the text into 16-bit machine code formatted for Vivado's $readmemh.
  • Export: Click "Download program.hex" and place the file in your simulation directory.
